# 東方非想天則 プロフィールメーカー

『東方非想天則』のプレイヤープロフィール画像をブラウザ上で作成できるツールです。
入力した内容を即座にキャンバスへ描画し、画像として保存できます。

### 使い方
1. `index.html` をブラウザで開きます。
2. フォームにプレイヤー名、段位、使用キャラなどの情報を入力します。
3. 「画像として保存」ボタンを押すと、PNG形式で画像がダウンロードされます。

### 注意事項
- このツールを動作させるには、プログラムと同じ階層に `image` フォルダ（各キャラクター画像）が必要です。
- 画像資産が含まれていない場合は、別途用意してパスを合わせる必要があります。
